trace

a network forensics tool

pinpoint your server...

This webtool is designed to be used when traceroute doesn't provide enough infomation. Why is your server lagging behind the others in speed? Why are you conencted to a very remote server? Figure it all out by using trace. The javascript implementation of traceroute. Note: The map automatically displays your current location, if location services are available.

This is not to be used for any illegal purposes.
Only to be used on networks you have explict permission to be testing! Only to be used to research server issues and/or locate servers. You take full responsibility by visiting this site.
